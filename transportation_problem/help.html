Справка

Программа для решения невырожденной транспортной задачи открытого или закрытого типа

Входные данные

Программа поддерживает числа с плавающей запятой для значений тарифов на перевозку грузов.

Чтобы выбрать файл с входными данными нажмите на "File" -> "Choose file" и в открывшемся диалоге выберите файл и нажмите "Ok"

Постановка задачи

После открытия входного файла в таблице появятся данные задачи - в заголовке таблицы производители P1 {мощность производства}, P2 {мощность производства} ... В первой колонке таблицы обозначены потребители в формате C1 {мощность потребления}, C2 {мощность потребления} ... Справа от таблицы появится надпись "task type: " {значение}. Значение "default" обзначает закрытая транспортная задача; "Over consuming" означает что задача открытого типа и мощность потребления превышает мощность производства; "Over production" означает что задача открытого типа и мощность производства превышает мощность потребления;

Получение решения

Для решения задачи нажмите на "Solve problem". В таблице появятся значения в квадратных скобках которые обозначают количество перевезенного груза из соответствующего производителя к потребителю. Общая стоимость наиболее отимального плана будет отображена под надписью с типом задачи.
